# صفحات HTML الأصلية

هذا المجلد يحتوي على صفحات HTML الأصلية للمشروع قبل تحويلها إلى Blade Templates.

## الملفات المتوفرة

- `index.html` - الصفحة الرئيسية الأصلية
- `about.html` - صفحة من نحن الأصلية
- `contact.html` - صفحة اتصل بنا الأصلية
- `service.html` - صفحة خدماتنا الأصلية
- `contracts-system.html` - صفحة نظام العقود الإلكترونية الأصلية
- `feature.html` - صفحة المميزات الأصلية
- `testimonial.html` - صفحة آراء العملاء الأصلية
- `pricing.html` - صفحة الأسعار الأصلية
- `blog.html` - صفحة المشاريع الأصلية
- `404.html` - صفحة الخطأ 404

## الاستخدام

يمكنك الرجوع إلى هذه الملفات للاستفادة من:
- التصميم الأصلي
- المحتوى الكامل
- البنية الهيكلية
- أي تفاصيل تصميمية قد تحتاجها

## ملاحظة

هذه الملفات هي نسخ احتياطية من التصميم الأصلي. الصفحات النشطة الآن موجودة في:
- `resources/views/pages/` - صفحات Blade الجديدة
